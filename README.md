# PLANETS API
***
### Performs planet operations:
- List all planets.
- Create a new Planet.
- Calculates the distance between two previous saved planets.

### Swagger URL:
http://localhost:8080/swagger-ui/index.html#

### Requiriments:
- Kotlin 1.6.21 +
- Java 11 +
- Maven 3.6.3 + 
- Spring Boot 2.7.1
